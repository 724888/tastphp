## Installation

```
git clone https://github.com/tastphp/tastphp.git

cd tastphp

composer update
```
## Features
* Symfony Style
* Dependency Injection Container.
* M（Service/Dao）VC Architecture
* Twig
* Console Support
* Debug Bar
* DBAL
* Migrations
* EventDispatcher
* More
